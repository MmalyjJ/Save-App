Методы раздела /user/

User response body:

{"uid":uid, "password":password, "phone":phone, "email":email,"balance":balance(long),"token":token,"day_created":day_created(Date object)}

Profile response body:

{"uid":uid, "first_name":first_name, "second_name":second_name, "birth_date":birth_date}

Portfolio response body:

{"uid":uid, "total_balance":total_balance(long), "actives":actives, "trade_history":trade_history}

Documents response body:

{"uid":uid, "passport_data":passport_data, "passport_issued":passport_issued, "passport_main_page":passport_main_page,"passport_registration_page":passport_registration_page,"is_accepted":is_accepted(boolean)}

  • /...

  • /new/ (Создание нового пользователя)

    Метод: GET
    Response type: User
    Параметры:
    • phone (not null) (string)
    • password (not null) (string)
    • email (string)
    • token (string)
    • balance (long)

  • /exists/ (Проверка наличия пользователя в БД)

    Метод: GET
    Response type: User
    Параметры:
    • phone (not null) (string)

  • /auth/ (Авторизация пользователя)

    Метод: GET
    Response type: User
    Параметры:
    • phone (not null) (string)
    • password (not null) (string)

  • /token/ (Валидация токена пользователя)

    Метод: GET
    Response type: User
    Параметры:
    • token (not null) (string)

  • /newPassword/ (Смена пароля пользователя)

    Метод: GET
    Response type: User
    Параметры:
    • token (not null) (string)
    • oldPass (not null) (string)
    • newPass (not null) (string)

  • /setProfile/ (Установить/обновить личные данные пользователя)

    Метод: GET
    Response type: Profile
    Параметры:
    • token (not null) (string)
    • first_name (not null) (string)
    • second_name (not null) (string)
    • bday (not null) (string)

  • /getProfile/ (Получить личные данные пользователя)

    Метод: GET
    Response type: Profile
    Параметры:
    • token (not null) (string)

  • /setDocuments/ (Указать данные удостоверения личности пользователя)

    Метод: GET
    Response type: Documents
    Параметры:
    • token (not null) (string)
    • doc_number (not required) (string)
    • issued (not required) (string)
    • main_page (not required) (string) - Base64 image
    • reg_page (not required) (string) - Base64 image

  • /getDocuments/ (Получить данные удостоверения личности пользователя)

    Метод: GET
    Response type: Documents
    Параметры:
    • token (not null) (string)

  • /getPortfolio/ (Получить данные портфолио пользователя)

    Метод: GET
    Response type: Portfolio
    Параметры:
    • token (not null) (string)
