Методы раздела /news/

{"nid":nid, "news_source":"news_source","category":"category", "body":"body", "news_date":news_date}

  • /getNews/ (Получить все новости)

    Метод: GET
    Response type: News
    Параметры:
    • token (not null) (string)

  • /getNewsCat/ (Получить новости по категории)

    Метод: GET
    Response type: News[]
    Параметры:
    • token (not null) (string)
    • category (not null) (string)
