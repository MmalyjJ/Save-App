Методы раздела /admin/

Admin response body:

{"phone":phone, "password":password, "email":email, "token":token}

  • /...

  • /auth/ (Авторизовать администратора)

    Метод: GET
    Response type: News
    Параметры:
    • token (not null) (string)

  • /getAllUsers/ (Получить всех пользователей)

    Метод: GET
    Response type: List<\User\>
    Параметры:
    • token (not null) (string)

  • /addUser/ (Добавить нового пользователя)

    Метод: GET
    Response type: User
    Параметры:
    • token (not null) (string)
    • phone (not null) (string)
    • password (not null) (string)
    • email (string)
    • balance (long)

  • /addAdmin/ (Добавить нового администратора)

    Метод: GET
    Response type: Admin
    Параметры:
    • phone (not null) (string)
    • password (not null) (string)
    • email (string)
    • secret (not null)(string) (Параметр уточнить у владельца сервера)

  • /deleteUser/ (удалить пользователя)

    Метод: GET
    Response type: User
    Параметры:
    • token (not null) (string)
    • uid (not null) (string)
